### Привет 👋
Выпускник магистратуры ВШЭ (программа «Искусственный интеллект») 2026 года.  
До магистратуры успешно окончил курс «Специалист Data Science plus» от Яндекс Практикум.  
📌Некоторые работы с описанием применяемых инструментов представлены в [портфолио](https://github.com/Dimentel/portfolio/blob/main/readme.md).

---

### 🛠️ Ключевые навыки

**Модели и алгоритмы**  
- Классический ML и нейросети — разработка, обучение, подбор гиперпараметров, понимаю устройство современных моделей  
- LLM: fine-tuning, instruction tuning, Speculative Decoding, промпт-инжиниринг, оптимизация инференса  
- RAG-системы с векторными базами данных  
- LLM-агенты и мультиагентные системы  
- NLP: классификация, NER, QA, поиск, генерация  
- Компьютерное зрение (CV): классификация, детекция, оценка возраста, генерация изображений  
- понимаю принципы RL и RLHF  
- Анализ данных, feature engineering, статистическое моделирование  
- Визуализация данных и результатов моделей с помощью современных инструментов аналитики  

**MLOps и инженерия**  
- End-to-end ML-пайплайны для для обучения, оценки и деплоя моделей  
- MLOps: MLflow, Airflow, DVC, Weights & Biases  
- Контейнеризация и оркестрация: Docker, Kubernetes  
- REST API: FastAPI, Django  
- Большие данные и распределённые вычисления: Spark, Hadoop  

**Инфраструктура**  
- Облачные платформы (Yandex Cloud, Immers.Cloud) для обучения и хранения данных (CPU, GPU)  

**Исследования в ИИ** 
-  чтение и воспроизведение современных статей
